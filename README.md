## Installation

```bash
# clone the repo
$ git clone https://github.com/zozo404/Weboops.git

$ cd Weboops

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# If that doesn't work, check if the node version is = 16.20
$ nvm list
$ nvm use 16.20

```
